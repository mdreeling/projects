Retrieving complete inventory by twitter username

Returns all the basic details for an entire users inventory including the name, model and a thumbnail image as base64.

Request:
GET
http://trackmygearfor.me/api/rest/inventory/search/{twitterusername}
Route Values:
{twitterusername}

The users twitter username (i.e @trackmygear = trackmygear)

Response:

Contains an array of items whose username attribute matches the twitterusername input parameter

Search for inventory items by twitter username

Adds a new template for an existing client by providing the name of the template and URLs for the HTML file and a zip of all other files.

Request:
GET
http://trackmygearfor.me/api/rest/inventory/search/{twitterusername}/{keyword}
Route Values:
{twitterusername}

The users twitter username (i.e @trackmygear = trackmygear)

{keyword}

The keyword to search on (Only the 'name' attribute of the gear is actually searched)

Response:

Contains an array of items whose 'name' contains the search string
