Architecture Diagrams

Architecture diagrams for both client and server.

Overall

 
Client

 

The client side code consists of several Javascript Modules

trackmygear.main - The main application
trackmygear.views - Backbone views
trackmygear.models - Backbone models
trackmygear.utils - Utility classes
trackmygear.config - Runtime configuration of REST urls and client side parameters

 
Server

 

The server side code consists of several Javascript Modules

dc-server-track - The main http server application
dc-server-api - RESTful api to MongoDB Server
common - Utility classes
config.mine - Runtime configuration of twitter API integration
